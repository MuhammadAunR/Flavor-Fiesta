# FlavorFiesta
FlavorFiesta is a fully responsive recipe web application built with React.js and Tailwind CSS. 
It provides users with a seamless browsing experience to discover, explore, and engage with a 
wide variety of recipes across multiple food categories.

---

## 🚀 Live Demo
[View Live](https://flavor-fiesta-byma.vercel.app/)

---

## 📋 About
FlavorFiesta was designed to bring the joy of cooking to everyone — from beginners to seasoned 
home cooks. The application features a curated collection of recipes organized by category, 
complete with detailed nutritional information, step-by-step cooking directions, and ingredient 
checklists. The goal was to build a clean, intuitive, and visually engaging platform that makes 
finding and following recipes as effortless as possible.

---

## ✨ Features
- 🎠 **Hero Carousel** — Auto-advancing image slider showcasing featured recipes with smooth slide transitions
- 🔍 **Search Bar** — Real-time recipe search that filters results instantly as you type
- 🏷️ **Category Filter** — Browse recipes by category including Meat, Breakfast, Vegan, Dessert, Lunch and Chocolate
- ❤️ **Favorites Filter** — Quickly surface recipes marked as favorites
- 📄 **Recipe Detail Page** — Each recipe includes full nutrition facts, ingredient checklists and step-by-step directions
- 📝 **Blog Page** — Informative articles and chef interviews for food enthusiasts
- 📬 **Subscribe Section** — Email subscription for recipe updates and cooking inspiration
- 📱 **Fully Responsive** — Optimized for all screen sizes from mobile to desktop

---

## 🛠️ Tech Stack
- **React.js** — Component-based UI architecture
- **Tailwind CSS** — Utility-first styling and responsive design
- **React Router DOM** — Client-side routing and navigation
- **AOS** — Animate on Scroll for smooth scroll-triggered animations
- **Lucide React** — Lightweight and consistent icon library
